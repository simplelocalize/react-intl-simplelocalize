# react-intl-simplelocalize

## Install

```bash
npm install --save react-intl-simplelocalize
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'react-intl-simplelocalize'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
